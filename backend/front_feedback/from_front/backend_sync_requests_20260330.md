**작성일:** 2026년 3월 30일

## 1. 신규 API 구현 요청

### 1.1 시술 가이드 챗봇 API (#159)
- **목적:** 관리자 대시보드 내 실시간 시술 가이드 제공
- **엔드포인트:** `POST /api/v1/admin/chatbot/ask/`
- **요청 데이터:** `{ "message": "질문 내용" }`
- **응답 데이터:** `{ "reply": "상세 시술 가이드 텍스트", "timestamp": "ISO-8601" }`
- **참고:** 상세 스펙은 같은 폴더의 `chatbot_api_request.md` 참조

### 1.2 설문 데이터 수집 API 확장
- **목적:** 성별에 따른 6개 설문 문항(q1~q6) 저장 지원
- **엔드포인트:** `POST /api/v1/survey/`
- **변경 사항:** 기존 필드 외에 남성/여성별로 분화된 6개 응답 값을 유연하게 수용할 수 있도록 모델 및 시리얼라이저 보강 필요

---

## 2. 기존 API 보완 및 버그 수정 요청

### 2.1 파트너 가입 시 상세 에러 응답 (#140)
- **요청 사항:** 가입 실패 시 전체 에러 메시지(`detail`) 외에, **필드별 에러(`errors`)** 정보를 상세히 내려주세요.
- **예시 구조:**
  ```json
  {
    "status": "error",
    "errors": {
      "phone": ["이미 등록된 전화번호입니다."],
      "business_number": ["사업자 번호 형식이 올바르지 않습니다."]
    }
  }
  ```

### 2.2 설문 페이지 성별 정보 전달
- **요청 사항:** `client_survey_page` 진입 시, 현재 세션에 저장된 고객의 `gender` 정보를 템플릿 변수(`client.gender`)로 확실히 전달해 주세요. (현재 프론트엔드 템플릿 분기 로직이 이에 의존하고 있음)

---

## 3. 인프라 및 기타
- **CORS 설정:** 로컬 테스트 환경(`localhost:8000`)에서 API 호출 시 CORS 에러가 발생하지 않도록 `django-cors-headers` 설정 점검 요청.
- **이미지 업로드:** `capture/upload/` API에서 세로형 이미지(3:4 비율) 수신 시 안면 인식 로직이 정상 작동하는지 재검증 요청.
