# [Backend Request] 관리자용 헤어시술 가이드 챗봇 API 구현 요청

## 1. 개요
관리자 대시보드 내에서 시술 방법을 실시간으로 문의할 수 있는 챗봇 기능을 위해 새로운 API 엔드포인트 구현이 필요합니다.

---

## 2. API 명세 (제안)

### 2.1 시술 가이드 질의 API
- **Endpoint:** `POST /api/v1/admin/chatbot/ask/`
- **Authentication:** `Bearer <Admin_Access_Token>` (관리자 권한 필수)
- **Request Body (JSON):**
  ```json
  {
    "message": "C컬 펌 시술 순서 알려줘"
  }
  ```
- **Response Body (JSON):**
  ```json
  {
    "reply": "C컬 펌 시술 순서는 다음과 같습니다:\n1. 샴푸 및 모발 진단...\n2. 연화제 도포...\n3. 와인딩...\n[주의] 손상모의 경우 온도를 120도 이하로 설정하세요.",
    "timestamp": "2026-03-30T15:30:00Z"
  }
  ```

---

## 3. 핵심 요구사항
1.  **시술 데이터 기반 답변:** AI 모델(LLM)이 우리 서비스의 헤어스타일 DB나 일반적인 미용 시술 가이드를 참고하여 전문적인 답변을 생성해야 합니다.
2.  **구조화된 텍스트 반환:** 프론트엔드에서 줄바꿈(`\n`)이나 리스트 형식을 처리할 수 있도록 응답 텍스트를 구조화해 주세요.
3.  **대화 맥락 유지 (선택 사항):** 가능하다면 간단한 세션 관리를 통해 이전 질문과 이어지는 답변이 가능하면 좋습니다.
4.  **에러 핸들링:** AI 모델 응답 지연 시 타임아웃 처리 및 적절한 에러 메시지(`detail`) 반환이 필요합니다.
